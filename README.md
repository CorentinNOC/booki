# Booki Starter Pack

## Contexte

Booki est une application web permettant aux utilisateurs de trouver des hébergements et des activités dans une ville donnée.

Ce projet a été réalisé dans le cadre de la formation Développeur Web chez OpenClassrooms.

L’objectif principal est d’intégrer une maquette en HTML et CSS en respectant les bonnes pratiques du développement web et le responsive design.

---

## Objectif

- Intégrer une maquette en HTML5 / CSS3
- Structurer une page web avec une sémantique claire
- Mettre en place un design responsive
- Utiliser Flexbox
- Respecter les standards W3C
- Versionner son projet avec Git & GitHub

---

## Stack technique

- HTML5
- CSS3
- Flexbox
- Git

---

## Compétences développées

- Intégration web
- Organisation du code
- Responsive design
- Gestion de version avec Git
- Respect d’une maquette client

---

## Résultats et impact

- Site responsive conforme à la maquette
- Code validé sans erreurs majeures
- Structure HTML propre et accessible
- Validation du projet par OpenClassrooms
